1. مقدمة HTML

HTML هي لغة ترميز النص الفائق، وهي اللغة القياسية المستخدمة لإنشاء صفحات الويب. تصف HTML هيكل صفحة الويب باستخدام مجموعة من العناصر والوسوم.

2. محررات HTML

يمكن كتابة HTML في أي محرر نصوص بسيط مثل Notepad، ولكن المحررات المخصصة للبرمجة توفر ميزات أفضل مثل تلوين الأكواد، الإكمال التلقائي، وتصحيح الأخطاء. من أشهر المحررات: Visual Studio Code, Sublime Text, Atom.

3. أمثلة أساسية

هذا هو الهيكل الأساسي لأي صفحة HTML.

<!DOCTYPE html>
<html>
<head>
  <title>عنوان الصفحة</title>
</head>
<body>
  <h1>عنواني الأول</h1>
  <p>فقرتي الأولى.</p>
</body>
</html>

4. العناصر (Elements)

عنصر HTML يتكون من وسم بداية، محتوى، ووسم نهاية. مثال: <p>هذا محتوى.</p>. بعض العناصر فارغة ولا تحتاج وسم نهاية، مثل <br> (فاصل أسطر) و <hr> (خط أفقي).

مثال:

<h2>العناصر</h2>
<p>هذه فقرة نصية.</p>
<br>
<p>وهذه فقرة أخرى بعد فاصل الأسطر.</p>

5. السمات (Attributes)

توفر السمات معلومات إضافية عن العناصر. تكتب دائمًا في وسم البداية على شكل `name="value"`. مثال: <a href="url"> و <img src="path">. السمة `disabled` لا تحتاج قيمة، مجرد وجودها يعطل العنصر.

مثال:

<!-- السمة href تحدد وجهة الرابط -->
<a href="https://www.google.com">اذهب إلى جوجل</a>

<!-- السمة src تحدد مسار الصورة -->
<img src="image.jpg" alt="وصف الصورة">

<!-- السمة lang تحدد لغة العنصر -->
<p lang="en">Hello World!</p>

6. العناوين

تستخدم العناوين من <h1> إلى <h6> لتحديد أهمية العناوين، حيث <h1> هو الأهم والأكبر حجمًا، و <h6> هو الأقل أهمية والأصغر حجمًا.

مثال:

<h1>هذا هو العنوان الأول</h1>
<h2>هذا هو العنوان الثاني</h2>
<h3>هذا هو العنوان الثالث</h3>
<h4>هذا هو العنوان الرابع</h4>
<h5>هذا هو العنوان الخامس</h5>
<h6>هذا هو العنوان السادس</h6>

7. الفقرات

يستخدم الوسم <p> لتعريف الفقرات النصية. المتصفح يضيف مسافة (هامش) قبل وبعد كل فقرة تلقائيًا لفصلها عن باقي المحتوى.

مثال:

<p>هذه هي الفقرة الأولى. تحتوي على نص يصف موضوعًا معينًا.</p>
<p>هذه هي الفقرة الثانية. لاحظ كيف يفصل المتصفح بينها وبين الفقرة الأولى.</p>

8. التنسيقات (Styles)

تستخدم سمة style لإضافة تنسيقات CSS مباشرة للعنصر (Inline CSS)، مثل الألوان والحجم والخطوط. الصيغة هي style="property:value;".

<p style="color:blue; font-size:16px;">هذه فقرة زرقاء.</p>
<h1 style="text-align:center; color:red;">عنوان في المنتصف باللون الأحمر</h1>

9. تنسيق النص

توفر HTML وسومًا لتنسيق النص بصريًا أو دلاليًا: <b> (عريض)، <strong> (مهم)، <i> (مائل)، <em> (مؤكد)، <mark> (تظليل)، <sub> (نص منخفض)، <sup> (نص مرتفع).

مثال:

<p>هذا نص <b>عريض</b> وهذا نص <strong>مهم</strong>.</p>
<p>هذا نص <i>مائل</i> وهذا نص <em>مؤكد</em>.</p>
<p>لا تنسَ شراء <mark>الحليب</mark> اليوم.</p>
<p>صيغة الماء هي H<sub>2</sub>O.</p>

10. الاقتباسات

<blockquote> للاقتباسات الطويلة التي تمثل قسمًا كاملاً، <q> للاقتباسات القصيرة داخل الفقرة، و <cite> لذكر اسم العمل المقتبس منه.

مثال:

<p>قال حكيم: <q>العلم نور والجهل ظلام.</q></p>

<blockquote cite="http://example.com">
  هذا اقتباس طويل من مصدر ما، يظهر عادةً مع مسافة بادئة في المتصفح.
</blockquote>

11. التعليقات

تكتب التعليقات بين <!-- و -->. لا تظهر في المتصفح وتستخدم لترك ملاحظات للمطورين في الكود أو لتعطيل جزء من الكود مؤقتًا.

مثال:

<!-- هذا تعليق، لن يظهر للمستخدم -->
<p>هذه الفقرة ستظهر في الصفحة.</p>

<!-- 
<p>هذه الفقرة لن تظهر لأنها داخل تعليق.</p>
-->

12. الألوان

يمكن تحديد الألوان بالاسم (red)، أو بقيم RGB، أو HEX، أو HSL. تستخدم مع خصائص CSS مثل color (لون النص) و background-color (لون الخلفية).

مثال:

<!-- استخدام اسم اللون -->
<p style="color:Tomato;">طماطم</p>

<!-- استخدام قيمة HEX -->
<p style="color:#FF5733;">برتقالي محمر</p>

<!-- استخدام قيمة RGB -->
<p style="background-color:rgb(200, 220, 255);">خلفية سماوية فاتحة</p>

13. الربط مع CSS

هناك 3 طرق لإضافة CSS: 1. Inline (باستخدام سمة style). 2. Internal (باستخدام وسم <style> في الـ head). 3. External (باستخدام وسم <link> لربط ملف css خارجي)، وهي الطريقة الأفضل للمشاريع الكبيرة.

مثال (External CSS):

<!-- ضع هذا السطر داخل وسم <head> -->
<link rel="stylesheet" href="styles.css">

14. الروابط

يستخدم الوسم <a> لإنشاء الروابط، وتحدد وجهة الرابط في السمة href. لجعل الرابط يفتح في تبويب جديد، استخدم target="_blank".

مثال:

<!-- رابط عادي -->
<a href="https://example.com">اذهب إلى Example.com</a>

<!-- رابط يفتح في تبويب جديد -->
<a href="https://google.com" target="_blank">افتح جوجل في تبويب جديد</a>

15. ألوان الروابط

يمكن التحكم في ألوان الروابط وحالاتها المختلفة باستخدام CSS. الحالات هي: a:link (الرابط العادي)، a:visited (تمت زيارته)، a:hover (عند التمرير)، a:active (عند النقر).

مثال (يوضع داخل وسم <style>):

a:link { color: blue; } /* الرابط الذي لم تتم زيارته */
a:visited { color: purple; } /* الرابط الذي تمت زيارته */
a:hover { color: red; } /* عند مرور الماوس فوق الرابط */
a:active { color: yellow; } /* عند النقر على الرابط */

16. الإشارات المرجعية

يمكن إنشاء رابط يقفز إلى جزء معين في نفس الصفحة عن طريق إعطاء ذلك الجزء id فريد، ثم استخدام href="#id-value" في الرابط.

مثال:

<!-- الرابط الذي سينقلك للأسفل -->
<a href="#section2">اذهب إلى القسم الثاني</a>

<!-- محتوى طويل هنا... -->
<div style="height: 1000px;"></div>

<!-- القسم المستهدف مع الـ id -->
<h2 id="section2">هذا هو القسم الثاني</h2>
<p>لقد وصلت إلى هنا!</p>

17. الصور

يستخدم الوسم <img> لإدراج الصور. السمة src تحدد مسار الصورة، و alt توفر نصًا بديلاً يظهر إذا فشل تحميل الصورة وهو مهم لإمكانية الوصول.

مثال:

<img src="https://placehold.co/400x200" alt="صورة توضيحية" width="400" height="200">

18. أيقونة الموقع (Favicon)

هي أيقونة صغيرة تظهر بجانب عنوان الصفحة في تبويب المتصفح. تضاف في الـ <head> باستخدام <link rel="icon" ...>.

مثال (يوضع داخل وسم <head>):

<link rel="icon" type="image/png" href="/favicon.png">

19. عنوان الصفحة

يحدد الوسم <title> داخل الـ <head> عنوان الصفحة الذي يظهر في شريط عنوان المتصفح أو تبويبه، وهو مهم جدًا لمحركات البحث.

مثال (يوضع داخل وسم <head>):

<head>
  <title>موسوعة HTML الشاملة</title>
</head>

20. عنصر Div

الوسم <div> هو حاوية عامة (container) تستخدم لتجميع عناصر أخرى معًا بغرض التنسيق أو التنظيم. هو عنصر من نوع Block (يأخذ السطر كاملاً).

مثال:

<div style="background-color:#f0f0f0; padding:15px;">
  <h2>عنوان القسم</h2>
  <p>محتوى هذا القسم موجود داخل حاوية div.</p>
</div>

21. Class و ID

class: يستخدم لتحديد فئة لواحد أو أكثر من العناصر لتطبيق نفس التنسيق. id: يستخدم لتحديد معرّف فريد لعنصر واحد فقط. لا يمكن أن يتكرر في نفس الصفحة.

مثال:

<style>
  /* استهداف العناصر عن طريق الكلاس */
  .city {
    background-color: tomato;
    color: white;
    padding: 10px;
  }
  /* استهداف العنصر عن طريق الـ ID */
  #mainHeader {
    border-bottom: 2px solid blue;
  }
</style>

<h1 id="mainHeader">قائمة المدن</h1>
<p class="city">القاهرة</p>
<p class="city">الرياض</p>

22. Iframes

يستخدم <iframe> لتضمين صفحة ويب أخرى داخل الصفحة الحالية. يمكن التحكم في أبعاده وخصائصه الأخرى عبر السمات.

مثال:

<iframe src="https://www.wikipedia.org" height="300" width="500" title="صفحة ويكيبيديا"></iframe>

23. JavaScript

يستخدم الوسم <script> لإضافة كود JavaScript إلى الصفحة، إما مباشرة داخل الوسم أو بربط ملف خارجي باستخدام السمة src.

مثال (تغيير محتوى نصي عند النقر):

<p id="demo">أهلاً بك!</p>

<button type="button" onclick="document.getElementById('demo').innerHTML = 'مرحباً بعالم JavaScript!'">انقر هنا</button>

24. مسارات الملفات

تستخدم لتحديد مواقع الملفات (صور، css). المسار النسبي (Relative) يكون نسبة لموقع الملف الحالي، والمسار المطلق (Absolute) يكون عنوان URL كامل.

مثال:

<!-- مسار مطلق (URL كامل) -->
<img src="https://www.w3schools.com/images/picture.jpg" alt="Mountain">

<!-- مسار نسبي (الصورة في نفس المجلد) -->
<img src="picture.jpg" alt="Mountain">

<!-- مسار نسبي (الصورة في مجلد الصور) -->
<img src="/images/picture.jpg" alt="Mountain">

25. عنصر Head

الـ <head> هو حاوية للمعلومات الوصفية (metadata) عن الصفحة. يحتوي على <title>, <meta>, <link>, <style>, <script>. محتواه لا يظهر في الصفحة مباشرة.

مثال:

<head>
  <meta charset="UTF-8">
  <meta name="description" content="دليل تعليمي للغة HTML">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>عنوان الصفحة</title>
  <link rel="stylesheet" href="style.css">
</head>

26. تخطيط الصفحة

يستخدم HTML5 وسومًا دلالية لتخطيط الصفحة مثل <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>. يتم تنسيقها باستخدام CSS Flexbox أو Grid.

مثال (هيكل صفحة بسيط):

<body>
  <header><h1>عنوان الموقع</h1></header>
  <nav><a href="#">الرئيسية</a></nav>
  <main>
    <section>
      <article>محتوى المقال هنا</article>
    </section>
  </main>
  <footer><p>حقوق النشر © 2025</p></footer>
</body>

27. التصميم المتجاوب

هو تصميم يجعل صفحة الويب تبدو جيدة على جميع الأجهزة. يعتمد بشكل أساسي على وسم <meta name="viewport"> واستخدام Media Queries في CSS لتطبيق تنسيقات مختلفة بناءً على عرض الشاشة.

مثال (Media Query في CSS):

<!-- هذا الكود يوضع داخل ملف CSS أو وسم <style> -->
/* للكمبيوتر */
.container { width: 80%; }

/* للهواتف (عندما يكون عرض الشاشة 600px أو أقل) */
@media (max-width: 600px) {
  .container { width: 100%; }
}

28. عناصر أكواد الكمبيوتر

تستخدم وسوم مثل <code> لعرض جزء من كود، <kbd> لمدخلات لوحة المفاتيح، و <pre> لعرض نص منسق مسبقًا مع الحفاظ على المسافات والأسطر.

مثال:

<p>جزء من الكود: <code>let x = 5;</code>.</p>
<p>اضغط <kbd>Ctrl</kbd> + <kbd>C</kbd> للنسخ.</p>
<pre>
  function sayHello() {
    alert("Hello!");
  }
</pre>
